# sentweet-postagger

### External JAR files
---------------
> * weka (version 3.8)
> * common-collections4-4.1
> * common-lang3
> * stanford-postagger-3.6.0
> * slf4j-api
> * slf4j-simple
> * LibSVM
> * libsvm

It is recommended to install the libsvm via Weka package manager (via **Tools** on Weka GUI) so that the compatibility of libsvm is guaranteed.
